# Cwebscanner_pro_max_ultra
Cwebscanner 加强版

使用命令
```
python Cwebscan.py x.txt -p 80,443 -t 50   -p：指定端口（默认会扫描常见web端口） -t：线程 x.txt：要扫描的目标文件
```

扫描结果会保存在当前目录下以扫描时间命名的csv文件中
